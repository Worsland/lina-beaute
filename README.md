# Plateforme de Réservation Beauté & Spa avec IA

## Description
Plateforme de réservation en ligne pour établissements de beauté avec assistant IA intelligent.

## Technologies
- **Frontend**: React.js, Redux, Material-UI
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **IA**: Google Gemini API
